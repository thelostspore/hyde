# Next Jekyll Blog
Starter Next.Js blog inspired by [Jekyll](https://jekyllrb.com/), based on [Hyde](https://github.com/pankajparashar/hyde.git). Makes it super-easy to migrate your posts and config from your old Jekyll-blog.

## How to install?
```
$ git clone
$ cd hyde
$ npm install
$ npm run dev
```
